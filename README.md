# Screenlit

Screenlit is an 'Artifact-First' AI Application Framework.

## Getting Started

### Installation

```bash
pip install screenlit
```

### Running an Application

You can run your Screenlit application using the CLI:

```bash
screenlit run app.py
```

Or by executing the script directly if it includes `sl.run()`:

```bash
python app.py
```
